Dog Breed Classifier

This is subtitle

This project was part of Udacity’s Artificial Intelligence Nanodegree. The goal of the project was to create a pipeline that takes an image and detects whether a human or dog is present, predicting the breed for the dog or deciding what dog breed the human looks similar to.

The core focus of the project was building the dog breed classifier. This was accomplished by adapting several pre-trained convolutional neural networks to a dog breed dataset and organizing these models into an ensemble. This was able to achieve 93.6% accuracy on 133 different dog breeds, with only ~70 example images per breed.

A full summary of this project is coming soon. In the meantime, please see the project’s github repository where the project can be viewed as a jupyter notebook.
